# <1장> C언어란?

C언어의 탄생
-1972년 데니스 리치(Dennis Ritchie)가 유닉스(Unix)를 개발하기 위해 개발한 프로그래밍 언어

C언어의 특징
1. 절차지향 언어
   ▷ 시간의 흐름에 따라 정해진 절차를 실행
2. 간결하고 효율적인 언어
   ▷ 재귀 호출이 가능
   ▷ 포인터와 메모리 관리 기능으로 세세한 부분까지  제어
   ▷ 크기와 메모리가 작아 실행 속도가 빠름
3. 이식성이 좋음
   ▷ 별다른 수정없이 다양한 운영체제의 플랫폼에서 제공되는 컴파일러로 번역해 실행.
   ▷ 다양한 CPU와 플랫폼의 컴파일러를 지원.
 4. 다소 어려움
   ▷ C언어는 좀 어렵지만 한 번 익히면 다른 프로그래밍 언어 공부에도 도움이 됨.

C 언어를 배워야 하는 이유
1. 많은 언어에 영향을 미친 가장 기본이 되는 언어
    ▷C 언어를 알면 그 이후의 프로그래밍 언어들도 습득이 매우 쉬움.
2. 현장에서 다양한 분야에 사용되는 범용적인 언어 
    ▷임베디드 시스템, 응용프로그램, 운영체제와 같은 시스템 소프트웨어 개발등 여러 분야에 널리 사용.
3. 프로그래밍 지식과 프로그래밍 방법을 학습
    ▷자료형부터 메모리 할당까지 프로그래밍에 필요한 지식을 학습함.

## 키워드
- 문법적으로 고유한 의미를 갖는 예약된 단어
- 프로그램 코드를 작성하는 사람이 이 단어들을 다른 용도로 사용금지.

## 식별자
- 프로그래머가 자기 스스로 정의해 사용하는 단어 (변수 이름, 함수 이름)
- 영문자, 숫자, 밑줄로 구성. 식별자의 첫문자로 숫자가 나올 수 없음.
- 키워드는 식별자가 될 수 없고, 대소문자를 구별하며, 중간에 공백문자를 포함하지 않음.
 
## 문장
- 프로그래밍 언어에서 컴퓨터에게 명령을 내리는 최소 단위
- 마지막에 세미콜론 ;으로 종료. (;를 빠뜨리면 문법오류 발생.)

## 블록과 들여쓰기
- 여러 개의 문자을 묶으면 블록, 블록 내부에서 문장들을 탭(tab)키로 오른쪽으로 들여쓰는 소스 작성 방식은 들여쓰기라고 한다.

```c
#include <stdio.h>

int main(void) 
{
  puts("C 언어");

  return 0;
}
```

들여쓰기가 잘된 코드

## 자료형의 분류 그림
- 기본형, 유도형, 사용자 정희형으로 나누어지고, 기본형은 정수형, 실수형, 문자형, void로 나누어짐.
- 프로그래머는 이러한 자료에 적당한 알고리즘을 적용해 프로그램을 작성.

## 변수
- 프로그래밍에도 정수와 실수같은 자료값을 저장하는 공간.
- 변수에는 고유한 이름이 붙여지며, 물리적으로 기억장치인 메모리에 위치
- 선언된 자료형에 따라 변수의 저장공간 크기와 저장되는 자료 값의 종류가 결정.
- 저장되는 값에 따라 변수 값은 바뀔 수 있고, 마지막에 저장된 하나의 값만 저장 유지.

```c
//자료형 변수이름
double height;

//여러 변수를 하나의 문장으로 선언
int width, height;
/*위 문장은 아래 문장과 같다.
* int width;
* int height;
*/
```
변수 선언문

## 대입
- 원하는 자료값을 선언된 변수에 저장
- 대입연산자 '=' : 오른쪽에 위치한 값을 왼쪽변수에 저장.

```c
int age;
age = 23;
age = 24;
// age란 변수에는 24라는 값이 저장됨.

//변수 초기화
int age = 20;
//변수 선언과 동시에 값을 저장.
```

## 정수 자료형
- 기본 키워드 int [4byte]
- short : int보다 작거나 같은 범위의 값 [2byte] 
- long : int보다 크거나 같은 범위의 값 [4byte]
- long long : long보다 큰 범위의 값 [8byte]
- signed : [부호가 있음] 음수, 0 양수를 모두 지원
- unsigned: [부호가 없음] 양수와 0만 저장.

*short = short int
 long = long int
 signed int = int

## 부동소수 자료형
- float : [4byte]
- double : [8byte]
- long double : [8byte]
- *float형 변수에 저장하려면

```c
float a = 3.14F;
```

double형 변수에 저장하려면

```c
double a = 3.14;
```

- 숫자 0을 정수 앞에 놓으면 8진수로 인식

- 실수는 e또는 E를 사용하여 10의 지수표현 방식으로 나타낼 수 있음.

## 심볼릭 상수 표현 방법
- const 상수 : 키워드 const
  1) 변수 선언 시 자료형 또는 변수앞에 놓임. 
  2) 선언 시 반드시 초기값 지정 필수, 선언이후  수정 불가능

```c
int const a = 3;
```

- 매크로 상수 : 전처리기 지시자 #define
  1) 매크로 상수를 정의하는 지시자
  2) 주로 대문자 이름으로 정의.
  3) 헤더파일 limits.h를 살펴보면 문자형과 정수형에 대한 다양한 매크로 상수가 정의됨.

```

- 열거형 상수 : 키워드 enum
  1) 정수형 상수 목록 집합을 정의.
  2) 열거형 상수에서 목록 첫 상수의 기본 값은 0, 다음부터 1씩 증가하는 방식.

```

## 입력함수 scanf()
- 형식지정열과 입력할 목록으로 구분
- scanf()는 보안문제로 경고메세지가 발생한다.

```c
//헤더 파일 위에 이 매크로상수를 정의하면 scanf()를 사용할 수 있다.
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>

void main() {
  int a, b;

  //형식지정열, 입력할 목록
  scanf("%d", &a);
  //입력할 목록에 있는 변수는 주소연산자&를 반드시 붙여야 된다.
  
  //scanf_s()를 주로 사용하는 것이 좋다.
  //숫자는 상관없지만 문자의 경우 입력할 변수 뒤에 크기를 명시해 줘야한다.
  //크기는 10,20,85같은 숫자를 넣으면 된다.
  //형식지정열, 입력할 목록, [크기]
  scanf_s("%d", &b);

  printf(a);
  printf(b);
}
```
